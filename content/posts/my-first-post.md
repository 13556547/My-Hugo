欢迎来到我的 Hugo 博客！

这是使用 ananke 主题创建的第一篇博客文章。Hugo 是一个快速、现代化的静态网站生成器，非常适合用于构建个人博客、文档网站等。

## 为什么选择 Hugo？

- **快速构建**: Hugo 可以在几秒钟内构建大型网站
- **简单易用**: 使用 Markdown 编写内容，无需复杂的配置
- **主题丰富**: 有大量精美的主题可供选择
- **部署方便**: 可以轻松部署到 Vercel、GitHub Pages 等平台

## 开始使用

要创建新文章，可以使用以下命令：

```bash
hugo new posts/my-new-post.md
```

然后编辑生成的 Markdown 文件，添加你的内容。

## 部署到 Vercel

本网站已经配置好 Vercel 部署，只需将代码推送到 GitHub，然后在 Vercel 中导入项目即可自动部署。

祝您使用愉快！
